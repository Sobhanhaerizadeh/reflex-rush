# ⚡ Reflex Rush

**Wait for green. Tap. Find out how fast you really are.**

A minimalist reaction-time game built with React and TypeScript.
The screen flips to green after a random delay — the instant it does, click as fast as you can.
Jump the gun and you start over.

## 🎮 How to play

1. **Click** anywhere to start.
2. The screen turns **red** — wait for it…
3. The moment it flips to **green**, click as fast as you can.
4. Your reaction time appears in **milliseconds**.
5. Beat your best — but don't click too early, or it's back to the start.

A good human reaction time sits around **200–250 ms**. Can you go lower?

---

## ✨ Features

- **Random delay** so you can never anticipate the green light
- **Precise timing** using `performance.now()` instead of `Date.now()`
- **Best-time tracking** across the session
- **"Too soon" detection** — clicking early sends you back
- **Keyboard support** (Space / Enter) and reduced-motion friendly
- **Zero dependencies** beyond React — one self-contained component

## 🚀 Getting started

You'll need [Node.js](https://nodejs.org) (version 18 or newer).

```bash
# 1. Clone the repository
git clone https://github.com/Sobhanhaerizadeh/reflex-rush.git
cd reflex-rush

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open the address shown in the terminal (usually `http://localhost:5173`).

To create a production build:

```bash
npm run build      # bundles into /dist
npm run preview    # serves the build locally to test it
```
